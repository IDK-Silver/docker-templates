# 簡單的網頁範例

## 使用說明


* 下載專案並建制
    ```
    git clone https://github.com/IDK-Silver/docker-templates
    cd docker-templates/website/apache_php_sql
    docker compose up -d
    ```
## 測試
* 打開網頁 (應該會看到 Hina 這一行字)
  
  http://localhost
* 資料庫管理網頁
  
  http://localhost:8080

## 端口(port)說明
* apache

  80, 443
* SQL

  3306

* phpmyadin

  8080
